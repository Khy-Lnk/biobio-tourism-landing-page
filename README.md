# Biobío Tourism Landing Page 

This is a static FrontEnd mockup for a tourism agency in the Biobío region, focused on delivering a clean User Interface (UI) using pure HTML and CSS. 

> **Note:** This project was developed as an academic requirement to practice web layout, semantic structuring, and styling without the use of JavaScript or external frameworks.

##  Features

* **Clean UI Design:** A visually appealing and organized layout tailored for a regional tourism agency.
* **Semantic HTML5:** Proper use of HTML tags to ensure a well-structured document.
* **Custom CSS Styling:** Fully styled from scratch, managing colors, fonts, and element positioning.
* **Static Mockup:** Designed as a visual prototype (buttons and links are visually represented but non-functional by design).

##  Technologies Used

* **HTML5:** Core page structure and content.
* **CSS3:** Visual styling, layouts, and aesthetics.

##  How to Run Locally

Since this is a static webpage, you don't need any server or special environment to view it.
